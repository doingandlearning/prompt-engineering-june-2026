# Module 2: Core Prompt Engineering

## Learning objectives

- Write developer-focused prompts that are clear, contextual, and constrained.
- Apply zero-shot, few-shot, and chain-of-thought prompting appropriately.
- Iteratively refine prompts when output is wrong or incomplete.
- Use these techniques in VS Code chat tools (Copilot, Codex, Claude).

## Session opening

- **Calibration (2 min):** Show-of-hands on current tool usage — sets pace for hands-on work; invites experienced users to contribute.

## BA track

The three Cs map directly to Jira tickets. Make this connection explicit before Module 3's Jira-to-code demonstration.
